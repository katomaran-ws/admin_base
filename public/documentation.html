isScrolledIntoView( )

True / false

Check if the element is currently shown in viewport.

isScrolledIntoView($(".footer-top"));

SetFullPageBanner( )

Set the Banner image full screen height.

$(window).resize(function(){ SetFullPageBanner(); });

$(window).load(function(){ SetFullPageBanner(); });

SetHalfPageBanner( )

Set the Banner image to 50% of screen height.

$(window).resize(function(){ SetHalfPageBanner(); });

$(window).load(function(){ SetHalfPageBanner(); });

custom_input_1(name, label, value, class_name)

Generates a custom input field with a label with different stylings

custom_input_1("title", "Title", "my title", "juro")

Valid class_name value's : "juro", "chisato", "manami", "nariko"
